**ComplexLab** is a modern desktop application designed to explore and solve mathematical problems involving **Complex Numbers**. It is the perfect tool for students and math enthusiasts who want to visualize math instead of just calculating it.



## ‚ú® Key Features
* **N-th Roots Calculator**: Find and visualize all the roots of any complex number instantly.

* **Futuristic UI**: A sleek "Cyberpunk" Dark Mode interface built with `CustomTkinter`.
* **Pro Math Rendering**: Uses **LaTeX** to display formulas clearly, just like in a textbook.
